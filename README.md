# AirGuard-STM32-ESP32-Air-Purifier
# AirGuard

AirGuard 是一款 **成品级智能空气净化器系统**，集成空气质量检测、风扇控制、本地屏幕交互以及云端 App 远程控制。  
本项目从 **硬件设计到软件实现全部由本人独立完成**，包括原理图、PCB、焊接、3D 外壳建模及嵌入式代码开发。

---

## 项目特点

- STM32 + ESP32 双 MCU 架构
- 空气质量实时监测（PM2.5 / PM10 / VOC / 温湿度等）
- 本地串口屏控制（Nextion）
- Blinker App 远程控制与状态同步
- 自研通信协议，状态强一致
- 3D 打印外壳，完整成品形态
